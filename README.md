# bspwm-dotfiles

## Instalar:

* bspwm
* sxhkd
* yay
* flashfocus
* picom 
* alacritty
* python-pywal
* feh
* rofi
* neofetch
* flameshot
* adapta-gtk-theme
* lxappearance
* LazyVim https://www.lazyvim.org/keymaps
* npm
* polybar

## Ejecutar

```
git clone https://github.com/AngelAld/bspwm-dotfiles.git

cd bspwm-dotfiles

cp -r config/* ~/.config/ 

cp -r misc/fonts/* ~/.local/share/fonts/ 

fc-cache -fv
```
